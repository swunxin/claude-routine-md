# 财经情报与策略分析助手 — 项目需求文档 v3.1

> **目标用户：** 通过中国大陆 QDII 类 ETF 投资美股的中长线投资者（仓位区间 30–60%）。  
> **核心理念：** 单纯的新闻聚合无法支持决策，市场是新闻、价格信号、政策动向、资金流向四个维度的合力。本工具将四者打通，输出可执行的策略判断，而非信息罗列。  
> **关键差异：** 因为持有的是 QDII ETF 而非美股本身，承担的是「底层资产 + 场内溢价」的双层风险。报告必须区分「美股事件」与「ETF 事件」（美股跌 ≠ ETF 跌，溢价可能反向变动）。  
> **创建日期：** 2026-05-31

---

## 1. 项目定位

### 1.1 工具要回答的问题

对于通过 QDII ETF 持有美股的中国大陆投资者，每个交易日开盘前需要快速回答：

- 过去 24 小时美股、油价、汇率、政策面发生了什么？
- 主导市场叙事是什么？是否在转变？
- 当前 ETF 溢价处于什么水平？是否进入危险区间？
- 如果美股回调 + 溢价收窄叠加，我会损失多少？
- 今天我应该加仓、减仓还是持仓不动？触发条件是什么？

### 1.2 三层处理架构

```
┌─────────────────────────────────────────────────────┐
│  层 1：多源数据采集                                  │
│  美股新闻 + 美股指标 + 政策事件 + 资金流              │
│  + QDII 溢价 + QDII 政策 + 人民币汇率                 │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  层 2：信号识别与权重分析                            │
│  主导叙事 + 关键指标变动 + 政策动向 + 异常告警        │
│  + 溢价水平评估 + 双层风险叠加测算                   │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  层 3：策略分析与情景输出                            │
│  情景概率 + 仓位建议 + 触发点 + 风险提示              │
│  + 双重触发逻辑（指数 + 溢价）+ 政策风险预警          │
└─────────────────────────────────────────────────────┘
```

- 汇率变动会同时影响净值和溢价方向

## 3. 层 2：信号识别与权重分析

### 3.1 主导叙事识别

每次运行时，识别**当前主导宏观叙事**（不超过 3 个）+ 1 个 QDII 资金面叙事：

| 叙事维度 | 示例 |
|----------|------|
| 通胀路径 | "PCE 粘性"vs"油价压制" |
| 央行动向 | "鸽派转向"vs"鹰派坐实"vs"按兵不动" |
| 地缘政治 | "停火预期"vs"冲突升级" |
| 经济增长 | "软着陆"vs"滞胀风险"vs"衰退临近" |
| 资金主线 | "AI 行情"vs"防御轮动"vs"价值回归" |
| **QDII 资金面** | **"额度紧张 / 申购暂停 / 监管警示"** |

### 3.2 关键指标变动判定

| 信号 | 阈值（24 小时） | 含义 |
|------|----------------|------|
| VIX 大幅上行 | +15% 以上 | 风险偏好骤降 |
| VIX 持续低位 | < 15 持续 5 日 | 市场过度乐观 |
| DXY 单日变动 | ±0.8% 以上 | 全球流动性变化 |
| 10Y 收益率 | ±10bp 以上 | 利率路径重定价 |
| 油价 | ±3% 以上 | 通胀变量重置 |
| 黄金 | +2% 以上 | 避险情绪激增 |
| 股债同跌 | 同日发生 | 滞胀风险定价 |
| 纳指 ETF 溢价单日扩大 | +2% 以上 | 场内情绪过热 |
| 纳指 ETF 溢价单日收窄 | -2% 以上 | 回归启动信号 |
| 人民币单日变动 | ±0.5% 以上 | QDII 净值显著传导 |

### 3.3 溢价水平评估（每次必出）

| 溢价区间 | 状态 | 含义 | 建议关注度 |
|----------|------|------|------------|
| < 1% | 🟢 正常 | 几乎无溢价风险 | 加仓友好 |
| 1–3% | 🟢 健康 | 流动性溢价的正常水平 | 加仓友好 |
| 3–5% | 🟡 偏高 | 需关注，但仍可承受 | 中性 |
| 5–8% | 🟡 警戒 | 溢价回归压力上升 | **谨慎加仓** |
| 8–12% | 🟠 危险 | 监管警示风险升高 | **建议减仓** |
| > 12% | 🔴 高危 | 历史上常触发剧烈收窄 | **强制减仓** |

### 3.4 政策与基本面动向

每次输出必须标注：
- 下一个关键事件距今天数（含 QDII 政策日历）
- 市场目前定价（如 CME FedWatch、Polymarket 隐含概率）
- 事件的潜在 surprise 方向（鸽派意外 / 鹰派意外）

### 3.5 异常信号告警

**美股侧：**
- 油价单日 ±5% 以上
- VIX 单日 +30% 以上
- 美联储官员"非常规"表态
- 单日股债同跌 1% 以上
- 信用利差快速扩大
- 美联储独立性受冲击

**QDII 侧：**
- 纳指 ETF 溢价单日 ±3% 以上
- 任一主流纳指 ETF 宣布暂停 / 限制申购
- 上交所 / 深交所对 ETF 溢价发布公开提示函
- 外管局公告新一轮 QDII 额度审批结果
- 人民币单日 ±0.8% 以上

### 3.6 双层风险叠加测算（核心）

每次报告必须输出**最坏情景的损失测算**：

```
实际损失 = 指数跌幅 + 溢价收窄幅度

情景测算：
- 温和回调：纳指 -5% + 溢价收窄 2% = 实际损失约 7%
- 中度回调：纳指 -10% + 溢价收窄 5% = 实际损失约 15%
- 深度回调：纳指 -15% + 溢价收窄 7% = 实际损失约 22%
- 政策黑天鹅：纳指 0% + 溢价收窄 8% = 实际损失约 8%
```

### 4.2 仓位建议（动态调整）

基于用户预设的**风险偏好**、**当前仓位**、**当前溢价水平**，给出建议：

| 输入 | 输出 |
|------|------|
| 当前仓位（%） | 建议加仓 / 减仓 / 持仓不动 |
| 浮盈浮亏 | 仓位结构调整建议 |
| 风险承受能力 | 加仓 / 减仓的具体触发点 |
| 当前溢价水平 | 是否适合加仓 / 是否需要减仓 |

**输出示例：**
> 用户当前 40% 仓位 + 持仓回报 40% + 当前溢价 7%（警戒区）。  
> **建议：** 维持 35–40% 仓位，不建议在溢价 > 5% 时加仓；等待"纳指回调 + 溢价回落至 3% 以内"的双重信号。

### 4.3 触发点清单（双重触发逻辑）

#### 加仓触发（双重条件）

| 纳指触发条件 | 溢价配合条件 | 加仓幅度 | 累计仓位 |
|--------------|--------------|----------|----------|
| 纳指跌 5% | **溢价 < 3%** | +5–8% | 40–46% |
| 纳指跌 7.6% | **溢价 < 3%** | +10% | 50–56% |
| 纳指跌 10% | **溢价 < 2%** | +10% | 60–66% |
| 纳指跌 15% | **溢价 < 2%** | +10% | 70–76% |

> **重要规则：** 若溢价高于 5%，即使纳指回调也**不建议加仓**。  
> 例外：纳指跌幅 > 15% 且溢价 < 5%，可启动逆势加仓。

#### 减仓触发（三类信号）

| 触发类型 | 触发条件 | 减仓幅度 |
|----------|----------|----------|
| 美股触发 | 纳指涨破前高 + VIX < 14 | -5% |
| 美股触发 | 10Y 突破 4.8% | -5% |
| 美股触发 | 单日股债同跌 1.5%+ | 先减 3% |
| 溢价触发 | 溢价 > 10% | -5% |
| 溢价触发 | 溢价 > 12% | -10% |
| 政策触发 | QDII 额度突然放开 | 立即减 5% |
| 政策触发 | 监管警示 ETF 溢价 | 立即评估减 5–10% |
| 政策触发 | 基金公司恢复 / 扩大申购 | 立即减 3–5% |

#### 强制不操作窗口

- 数据公布前后 48 小时（非农、CPI、PCE、FOMC）
- 外管局季度 QDII 额度公告前后 24 小时

### 4.4 风险提示（强制输出）

## 6. 关键监测指标详解

### 6.1 纳指 100（^NDX）— 核心标的

**为什么权重最高？**
- QDII 类 ETF 的跟踪标的，直接决定底层净值
- AI 主题、科技龙头集中

**关注口径：**
- 即时点位 + 5 日均值
- 与 S&P 500 的相对强弱
- 主要成分股动向（NVDA、MSFT、AAPL、AMZN、GOOGL、META、TSLA）

### 6.2 ETF 溢价 — 与纳指并列重要

**为什么权重最高？**
- 对 QDII 持有者，溢价 vs 净值的 1% 变化 = 实际损益 1%
- 比 VIX 更直接影响盈亏
- 政策事件触发的非线性风险

**关注口径：**
- 当日溢价 + 5 日均值 + 月内极值
- 与同类基金（159941、513300）的对比
- 申购暂停 / 恢复状态

**机制详解：**

```
QDII 额度受限（来自外管局）
    ↓
基金公司可能暂停或限制申购
    ↓
场内供应有限 + 场外需求旺盛
    ↓
场内价格高于净值 → 形成溢价
    ↓
溢价过高时触发监管警示或基金公司恢复申购
    ↓
溢价快速收窄 → 持有者额外损失
```

**预估值输出示例：**
- 当前溢价：7%（警戒区）
- 7 日预估：5–9%（中等置信度）
- 触发收窄的可能事件：基金公司公告恢复申购（30% 概率） / 监管警示（20% 概率） / 美股大跌带动场内抛盘（25% 概率）

### 6.3 油价（Brent / WTI）— 通胀变量之首

**为什么权重高？**
- 每 10% 油价变动 → 整体 PCE 通胀 ±0.2pp
- 直接影响美联储路径
- 反映地缘风险

**关注口径：**
- 即时价格 + 5 日均价 + 月内涨跌幅
- 与 OPEC+ 产量、SPR 释放速度的对照
- EIA 周报（每周三）

### 6.4 美元指数（DXY）

**为什么权重高（双重含义）？**
- **全球层面：** DXY 强 → 新兴市场资金外流；DXY 弱 → 黄金、商品多头
- **QDII 层面：** DXY 强 = 美元升值 = 人民币贬值压力 = QDII 净值上升；DXY 弱反之
- **关键：** 汇率传导路径是 QDII 持有者特有的，不能忽略

**关注口径：**
- 当前位置 + 关键阻力位（如 100.2、102.5）
- 与人民币中间价的联动

### 6.5 VIX 恐慌指数

**为什么权重重要？**
- VIX < 15 持续 → 风险偏好极致，反弹空间有限
- VIX > 25 → 历史买点机会
- 反映期权市场对未来 30 天波动率的隐含定价

**对 QDII 用户的特别说明：**
- VIX 仅是间接信号，对持有者的实际损益影响**比溢价水平次要**
- 但 VIX 跳升常常伴随溢价收窄，因此仍需监测

### 6.6 10Y 国债收益率

**为什么权重高？**
- 直接影响股票估值（DCF 分母）
- 反映期限溢价 + 通胀预期
- 反映财政可持续性担忧

**关键阈值：**
- 突破 4.8% = 股市承压
- 跌破 4.2% = 利好成长股

### 6.7 EIA 原油库存

**为什么权重需要提升？**
- 商业库存动态影响后续油价走势
- SPR 释放速度反映政策意图
- 库存数据领先油价 2–4 周

**关注口径：**
- 每周三 EIA 报告
- 商业库存 + SPR 同步监测
- 西海岸（PADD 5）单独关注（对中东依赖度最高）

### 6.8 USD/CNY 汇率

**为什么权重需要提升？**
- QDII 净值 = 美股市值（美元计价）× 汇率（USD/CNY）
- 汇率波动 1% = QDII 净值波动 1%（在其他不变情况下）

**关注口径：**
- 即期汇率 + 离岸 CNH
- 央行中间价（政策意图信号）
- 5 日 / 月内波动幅度

**预估值输出示例：**
- 当前 USD/CNY 假设 7.25
- 若美联储鸽派：人民币升值至 7.15 → QDII 净值 -1.4%
- 若美联储鹰派：人民币贬值至 7.35 → QDII 净值 +1.4%

### 6.9 QDII 政策动向

**为什么权重高？**
- 是溢价收窄的最大单一触发因素
- 历史上多次发生"政策出 → 溢价崩"

**关注口径：**
- 外管局季度 QDII 额度公告
- 证监会、上交所 / 深交所 ETF 溢价警示
- 基金公司申赎状态公告

## 7. 策略分析输出范式

### 7.1 数据 + 预估值的组合

每个关键指标必须同时输出：
- **当前值**（事实）
- **预估值**（推断，明确标注）
- **历史参照**
- **置信度**（高 / 中 / 低）

**示例：**

> **513100 国泰纳指 ETF 溢价：**
> - 当前：7.0%（警戒区）
> - 1 周预估：5–9%（中等置信度）
> - 极端情景：若基金公司宣布恢复申购 → 1–2 日内收窄至 1–3%（低概率但高冲击）
> - 历史参照：2024 年 11 月、2025 年 3 月 / 7 月各发生过一次 10%+ 溢价快速收窄事件

### 7.2 双层情景树

```
当前节点：纳指 = 26,972 + 溢价 = 7%

第一层（美股）：
    ├─ 情景 A（50%）：浅度回调 -3% to -5%
    ├─ 情景 B（30%）：中度回调 -7% to -10%
    └─ 情景 C（15%）：深度回调 -15% to -20%

第二层（溢价，独立于美股）：
    ├─ 情景 α（55%）：溢价维持 5–9%
    ├─ 情景 β（25%）：溢价收窄至 2–4%
    └─ 情景 γ（20%）：溢价跳升至 10%+

复合情景（举例）：
- A × α（27.5%）：温和回调 + 溢价维持 → 实际损失约 4%
- B × β（7.5%）：中度回调 + 溢价收窄 → 实际损失约 13%（双杀）
- A × γ（10%）：温和回调 + 溢价扩大 → 实际损失约 1%（溢价缓冲）
```

### 7.3 反身性提醒

每次报告必须包含一段"市场已经定价了什么 vs 真正的预期差在哪里"的分析：

> **市场已经定价（美股侧）：**
> - 美伊协议 6 月签署（约 70% 概率）
> - 美联储 6 月维持利率（99% 概率）
> - AI 资本支出继续高增长
>
> **市场可能未充分定价（美股侧）：**
> - 原油库存创 23 年新低带来的供应反弹风险
> - SPR 释放停止后油价的二次冲击
>
> **QDII 侧未充分定价：**
> - 当前溢价 7%（警戒区）的回归压力
> - 外管局新一轮 QDII 额度审批可能放开
> - 人民币汇率政策的不确定性
> - 监管警示 ETF 溢价的潜在时点

## 11. 设计哲学

这个工具不是为了**取代**用户的判断，而是为了**支持**判断：

1. **数据先于观点。** 每个判断必须能追溯到具体数据。
2. **概率优于断言。** 用"30% 概率"替代"会 / 不会"。
3. **触发点优于预测。** 具体触发条件比"未来会涨/跌"更有用。
4. **承认无知。** 明确标注"已知未知"和"未知未知"边界。
5. **反共识思维。** 当市场一致预期时，提示反向风险。
6. **双层视角。** 美股事件 ≠ QDII 持仓事件；溢价是独立风险维度。
7. **政策敏感。** 中国大陆投资者面对的政策风险不能被忽略。

**最终目标：** 让用户在每个交易日开盘前 10 分钟，就能读完报告并做出该日的仓位决策——既不被美股原生指标误导，也不被中国场内的过热情绪带偏。
