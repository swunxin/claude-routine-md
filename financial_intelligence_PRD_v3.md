# 财经新闻摘要与策略分析助手 — 项目需求文档 v3.0

> **版本说明：** 本文档由 v2.0「数据 + 信号 + 策略三层架构」升级为 v3.0「QDII 视角的双层风险分析框架」。  
> **v3.0 核心理念升级：** 中国大陆投资者通过 QDII 类 ETF 间接持有美股，承担的是**底层资产 + 场内溢价**的双层风险。原 v2.0 的指标体系（聚焦 S&P 500 / VIX / DXY）对直接持有美股有效，但对 QDII 持有者**遗漏了最大的特异性风险**——溢价回归、QDII 政策、汇率传导。  
> **创建日期：** 2026-05-31 | **目标用户：** 中等仓位（30–60%）、通过 QDII 类 ETF 投资美股的中长线投资者。

---

## 1. 项目重新定位

### 1.1 v2.0 的局限

v2.0 假设用户**直接持有美股**，所以指标体系聚焦于：
- 美股原生指数（S&P 500、纳指、道指、罗素）
- 美元 / 利率 / VIX
- 美股相关大宗商品（油、金）

但对中国大陆投资者，**这些指标不足以覆盖真实风险**。一个典型案例：

> **场景：** 美股大涨 + 纳指 ETF 溢价从 3% 扩大到 10%  
> **v2.0 解读：** 持仓升值，可能考虑减仓锁利  
> **v3.0 解读：** 持仓升值 = 指数涨 + 溢价扩大，但**溢价已进入危险区间**，应立即减仓——不是因为美股见顶，而是因为溢价回归风险升高

v2.0 完全无法识别上述差异。这是 v3.0 必须解决的核心问题。

### 1.2 v3.0 的架构升级

**保留 v2.0 的三层架构**（数据 → 信号 → 策略），**在每一层加入 QDII 维度**：

```
┌─────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────┘
                        ↓

│  层 2：信号识别与权重分析                            │
│  主导叙事 + 关键指标变动 + 政策动向 + 异常告警        │
│  【v3 新增】+ 溢价水平评估 + 双层风险叠加测算         │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  层 3：策略分析与情景输出                            │
```

**汇率传导机制：**
- 人民币贬值 → QDII 净值上升（即使纳指不动）→ ETF 场内价随之上升
- 人民币升值 → QDII 净值下降 → ETF 场内价承压
- 因此，**汇率变动会同时影响净值和溢价方向**

### 3.2 关键指标变动判定（保留 v2.0 + v3 新增）

### 3.3 【v3 新增】溢价水平评估

**每次报告必须输出当前溢价的状态分级：**

| 溢价区间 | 状态 | 含义 | 建议关注度 |
|----------|------|------|------------|
| < 1% | 🟢 正常 | 几乎无溢价风险 | 加仓友好 |
| 1–3% | 🟢 健康 | 流动性溢价的正常水平 | 加仓友好 |
| 3–5% | 🟡 偏高 | 需关注，但仍可承受 | 中性 |
| 5–8% | 🟡 警戒 | 溢价回归压力上升 | **谨慎加仓** |
| 8–12% | 🟠 危险 | 监管警示风险升高 | **建议减仓** |
| > 12% | 🔴 高危 | 历史上常触发剧烈收窄 | **强制减仓** |

### 3.4 政策与基本面动向

每次输出标注：
- **下一个关键事件**（如"6 月 5 日 5 月非农"）距今天数
- **市场目前定价**（如"CME FedWatch 显示 6 月维持利率概率 99%"）
- **该事件的潜在 surprise 方向**（鸽派意外 / 鹰派意外）
- **【v3 新增】QDII 政策日历**（外管局例会、申购恢复公告等）

### 3.5 异常信号告警

**美股侧（保留 v2.0）：**
- 油价单日 ±5% 以上
- VIX 单日 +30% 以上
- 美联储官员"非常规"表态
- 单日股债同跌 1% 以上
- 信用利差快速扩大
- 美联储独立性受冲击

**【v3 新增】QDII 侧：**
- 纳指 ETF 溢价单日 ±3% 以上
- 任一主流纳指 ETF 宣布暂停 / 限制申购
- 上交所 / 深交所对 ETF 溢价发布公开提示函
- 外管局公告新一轮 QDII 额度审批结果
- 人民币单日 ±0.8% 以上

### 3.6 【v3 新增】双层风险叠加测算

每次报告输出**最坏情景的损失测算**：

```
实际损失 = 指数跌幅 + 溢价收窄幅度

情景测算：
- 温和回调：纳指 -5% + 溢价收窄 2% = 实际损失约 7%
- 中度回调：纳指 -10% + 溢价收窄 5% = 实际损失约 15%
- 深度回调：纳指 -15% + 溢价收窄 7% = 实际损失约 22%
- 政策黑天鹅：纳指 0% + 溢价收窄 8% = 实际损失约 8%
```

## 4. 层 3：策略分析与情景输出

### 4.1 情景概率评估（保留 v2.0 + v3 新增维度）

每次输出基于**当前数据**给出三种情景的概率分布：

| 情景 | 触发条件 | 概率（估算） | 资产含义（QDII 视角） |
|------|----------|--------------|------------------------|
| 鸽派 | 通胀降温 + 就业走弱 + 美联储释放鸽派 | XX% | 纳指多头 + 溢价可能扩大（需警惕） |
| 鹰派 | 通胀粘性 + 就业强劲 + 美联储紧缩 | XX% | 纳指承压 + 溢价可能收窄（双杀风险） |
| 横盘 | 数据混乱 + 政策含糊 | XX% | 防御性配置 + 监控溢价 |

**【v3 新增】QDII 特定情景：**

| 情景 | 触发条件 | 概率 | 含义 |
|------|----------|------|------|
| 溢价收窄 | QDII 额度放开 / 申购恢复 / 监管警示 | XX% | 即使美股不跌，ETF 也会跌 |
| 溢价扩大 | 美股大涨 + 申购维持限制 | XX% | 持仓账面增值但风险累积 |
| 汇率冲击 | 人民币意外贬值 / 升值 | XX% | 净值反向传导，需重估持仓 |

### 4.2 仓位建议（动态调整）

基于用户预设的**风险偏好**、**当前仓位**、**当前溢价水平**，给出建议：

| 输入 | 输出 |
|------|------|
| 当前仓位（%） | 建议加仓 / 减仓 / 持仓不动 |
| 浮盈浮亏 | 仓位结构调整建议 |
| 风险承受能力 | 加仓 / 减仓的具体触发点 |
| **【v3 新增】当前溢价水平** | **是否适合加仓 / 是否需要减仓** |

**v3 示例输出：**
> 用户当前 40% 仓位 + 持仓回报 40% + 当前溢价 7%（警戒区）。  
> **建议：** 维持 35–40% 仓位，不建议在溢价 > 5% 时加仓；等待"纳指回调 + 溢价回落至 3% 以内"的双重信号。

### 4.3 触发点清单（v3 升级 — 双重触发逻辑）

**每次输出具体可执行的触发点：**

#### 加仓触发（双重条件 — v3 关键升级）

| 纳指触发条件 | 溢价配合条件 | 加仓幅度 | 累计仓位 |
|--------------|--------------|----------|----------|
| 纳指跌 5% | **溢价 < 3%** | +5–8% | 40–46% |
| 纳指跌 7.6% | **溢价 < 3%** | +10% | 50–56% |
| 纳指跌 10% | **溢价 < 2%** | +10% | 60–66% |
| 纳指跌 15% | **溢价 < 2%** | +10% | 70–76% |

> **v3 重要规则：** 若溢价高于 5%，即使纳指回调也**不建议加仓**。  
> 例外：纳指跌幅 > 15% 且溢价 < 5%，可启动逆势加仓。

#### 减仓触发（三类信号）

| 触发类型 | 触发条件 | 减仓幅度 |
|----------|----------|----------|
| 美股触发 | 纳指涨破前高 + VIX < 14 | -5% |
| 美股触发 | 10Y 突破 4.8% | -5% |
| 美股触发 | 单日股债同跌 1.5%+ | 先减 3% |
| **【v3 新增】溢价触发** | 溢价 > 10% | -5% |
| **【v3 新增】溢价触发** | 溢价 > 12% | -10% |
| **【v3 新增】政策触发** | QDII 额度突然放开 | 立即减 5% |
| **【v3 新增】政策触发** | 监管警示 ETF 溢价 | 立即评估减 5–10% |
| **【v3 新增】政策触发** | 基金公司恢复 / 扩大申购 | 立即减 3–5% |

#### 强制不操作窗口

- 数据公布前后 48 小时（非农、CPI、PCE、FOMC）
- **【v3 新增】**外管局季度 QDII 额度公告前后 24 小时

### 4.4 风险提示（强制输出 — v3 扩展）

每次报告底部必须包含：

1. **本次报告最大的"已知未知"**
2. **市场目前定价不足的尾部风险**
3. **未来 7 天内可能改变叙事的事件**

### 6.2 ETF 溢价（v3 新增 — 与纳指并列重要）

**为什么权重最高？**
- 对 QDII 持有者，**溢价 vs 净值的 1% 变化 = 实际损益 1%**

**机制详解：**

```

    ↓
场内供应有限 + 场外需求旺盛

场内价格高于净值 → 形成溢价
    ↓
溢价过高时触发监管警示或基金公司恢复申购
    ↓

```

**预估值输出示例：**

## 12. 设计哲学（v3 强化）

这个工具不是为了**取代**用户的判断，而是为了**支持**判断：

1. **数据先于观点。** 每个判断必须能追溯到具体数据。
2. **概率优于断言。** 用"30% 概率"替代"会 / 不会"。
3. **触发点优于预测。** 具体触发条件比"未来会涨/跌"更有用。
4. **承认无知。** 明确标注"已知未知"和"未知未知"边界。
5. **反共识思维。** 当市场一致预期时，提示反向风险。
6. **【v3 新增】双层视角。** 美股事件 ≠ QDII 持仓事件；溢价是独立风险维度。
7. **【v3 新增】政策敏感。** 中国大陆投资者面对的政策风险不能被忽略。

**最终目标：** 让用户在每个交易日开盘前 10 分钟，就能读完报告并做出该日的仓位决策——既不被美股原生指标误导，也不被中国场内的过热情绪带偏。
