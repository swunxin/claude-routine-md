# M3 加减仓策略（每月回顾）

**主引擎 = 周线主开关 + 日线二级阶梯；溢价仅作卖出侧 + 冷启动重建的薄覆盖层。**
依据：溢价对 QDII 持有人不对称——帮卖（兑现虚高场内价）、坑买（高位追入承担回归风险）。

## M3.1 核心结构与铁律

| 层级 | 信号 | 作用 |
|------|------|------|
| 周线 | 周线趋势 / 周均线 | **总开关**：走好 = 可持仓；走坏 = 全面退出、释放底仓 |
| 日线 | 日线 + 日 20MA | 战术节奏：在仓位区间内调节 |
| 仓位区间 | — | 底仓 **25%（神圣）** ── 上限 100%（常规）/ 75%（周线末期） |

> **铁律：25% 底仓神圣，唯一释放钥匙 = 周均线走坏。任何溢价水平（即使 >12%）都不得击穿 25% 底仓。**

## M3.2 减仓（价格驱动，溢价不设限）

| 步骤 | 触发 | 仓位降到 |
|------|------|----------|
| 一级（预防） | 涨幅充分 + 日线走差 | 50% |
| 二级（确认） | 跌破日 20MA | 25%（底仓） |
| 终极 | 周均线走坏 | 跌破底仓，清仓 / 留最小观察仓 |

**叠加的卖出侧覆盖层（均在底仓之上操作）：**
- 溢价 >10% → 减 5%；>12% → 减 10%
- 政策：QDII 额度放开 → 立即减 5%；监管警示溢价 → 评估减 5–10%；基金恢复/扩大申购 → 减 3–5%
- 宏观：纳指破前高 + VIX<14 → -5%；10Y 破 4.8% → -5%；单日股债同跌 1.5%+ → 先减 3%

## M3.3 加仓（两类，区别对待溢价）

| 类型 | 场景 | 触发 | 溢价闸门 |
|------|------|------|----------|
| A 战术加回 | 减仓后日线未破 20MA 且反弹 | 回到减仓前档位 | 无（回踩已压掉溢价） |
| B 冷启动重建 | 从底仓 / 空仓、高位重新建仓 | 趋势确认 | **溢价 < 5% 才建** |

> B 闸门设 5%（非 3%）：513100 自 2024-11-25 暂停申购、溢价结构性偏高，<3% 几乎等不到。
> 机会性补充：周线仍向好却闪崩时，纳指急挫 -10%/-15% 且溢价 <3% → 机会性逆势加仓（非主引擎）。

# M4 投资观点与心理边界（季度 ~ 半年回顾）

## M4.1 主导观点

1. 美股长周期偏多（赌 AI 资本支出周期 + 长期向好）
2. 下跌形态偏"震荡做周线头 → 压合周均线 → 再突破"，而非 V 型崩盘
3. 6 月偏震荡（高位 + 外部动荡，推高动力不足）；鸽派概率偏高（赌特朗普主动制造催化剂）
4. 对溢价回归风险敏感；理解 EIA 库存的领先意义

**关注优先级（★★★★★）：** ETF 溢价 + QDII 政策、纳指 100、油价/地缘、通胀/美联储。次级：VIX、美元/利率、标普。

## M4.2 心理边界

| 边界 | 状态 |
|------|------|
| 浮盈回吐到 0% | 边缘（可能情绪化决策） |
| 浮盈回吐 50% | 可承受 |
| 单日溢价收窄 1–2% | 已经历，可承受 |
| 单日溢价收窄 5%+ | 未经历，存在心理冲击 |
| 指数 + 溢价双杀（合计 10%+） | 未经历，最大尾部心理风险 |

**决策偏好：** 倾向"稳定主仓 + 分批加仓"、接受小幅减仓锁利、重纪律避免决策疲劳；不做"先减后加"（自评难控）、不做短线/频繁操作。

## M4.3 已踩坑教训

- **三步减仓 50%→25%→1%（2026-06-09/10）击穿了 25% 底仓。** 溢价顾虑（记忆 >10%，日志 6.2%，存疑）越权动了本应只归周线管的底仓，事后纳指连涨踏空。教训：①"底仓神圣"此前未写入、压力下最先被破，现已显式写入 M3.1；②用 ETF 口径（场内当日 +1.5%）而非纳指口径（+3.5%）衡量盈亏与懊悔。
- 待补强的盲点：对单一二手数据点的警惕、对鸽派情景持续性的可能高估。

# M5 基本盘与沟通约束（极少变动）

## M5.1 背景

中长线投资者（看周线/月线），通过中国大陆 QDII 类 ETF 投资美股；有经验、已建立独立分析框架；风险承受中等。

## M5.2 红线（绝不建议的方向）

杠杆/期权/衍生品激进策略、单一个股重仓、短线/日内、加密货币具体操作、A 股/港股个股推荐（港股 ADR 参考除外）、QDII 套利（不博溢价价差）。

## M5.3 沟通约束（PRD 已含全局规则，此处仅记用户偏好）

- 风格：数据 + 概率 + 触发点三层；带反共识视角；明确区分"事实/推断"；中文结构化（表格 + 分级标题）；避免情绪化词与冗长铺垫。
- 回应方式：用户提判断时先肯定对的部分再压力测试；表达情绪时识别但不放大、回到数据；数字存疑时给区间 + 来源差异。
- 强约束：不替用户决策、不预测精确点位/时点、每次标注最大"已知未知"、所有数字带时间戳、不忽视溢价维度、区分美股事件 vs ETF 事件。
